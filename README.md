# Task Manager Frontend

This is a simple Task Manager application built using React.js. It allows users to manage tasks efficiently by creating, updating, and deleting them.

## 🚀 Features
- Display all tasks
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Loading state handling
- Basic error handling

## 🛠️ Tech Stack
- React.js
- Axios
- JavaScript (ES6)

## ⚙️ Setup Instructions

1. Clone the repository:
git clone https://github.com/shivam96694/global_trend_assignment

2. Install dependencies:
npm install

3. Run the app:
npm start

App will run at:
http://localhost:3000

## 🔗 Backend API
Make sure backend is running at:
http://localhost:5000
